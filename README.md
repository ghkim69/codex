## AI 기반 인력운영 최적화 시스템

MES 실시간 생산정보, 작업실적, 공정별 작업부하, 작업자 숙련도, 근무 가용정보를 통합 분석하여 생산공정별 최적 인력배치안을 자동 산출하는 AI 기반 인력운영 최적화 시스템의 설계 문서를 정리했습니다.

### 문서 구성
- [AI 기반 인력운영 최적화 시스템 설계서](docs/ai-workforce-optimization-system.md)

### 핵심 설계 포인트
- MES/ERP/HR/근태 데이터를 통합한 실시간 의사결정 구조
- 공정 부하 예측 + 작업자 역량 스코어링 + 수리최적화 결합
- 생산성 향상, 병목 최소화, 설명 가능한 배치 추천 지원
